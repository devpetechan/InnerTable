A tinted pill that shows a friend's or a place's take: Want to Try, Recommends, Hard Pass, or the neutral outline "Mixed" for split signals.

```jsx
<StatusChip status="recommends" />
<StatusChip status="try" />
<StatusChip status="pass" />
<StatusChip status="mixed" />
```

Variants: `try` (blue), `recommends` (green), `pass` (red), `mixed` (neutral outline, no dot — used on the card header when a place has both recommends and hard-passes). Pass `label` to override copy.

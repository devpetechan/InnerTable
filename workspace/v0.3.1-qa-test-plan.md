# v0.3.1 (REL-19) — Visual Redesign QA Test Plan

Manual QA for IT-088. Run top to bottom after IT-084–087 land. Format mirrors
`workspace/phase-2-4-test-plan.md`.

## Setup (one-time)

1. Start a local server in the repo root: `python3 -m http.server 8080` and open
   `http://localhost:8080`.
2. Add `http://localhost:8080` to Supabase → Authentication → Redirect URLs.
3. Keep DevTools console open (F12). Test in **Chrome** and **Safari**.
4. Repeat key layout checks at a **mobile viewport** (375×812 or iPhone preset).

## Known limitations — do NOT count as failures

- **Filter & sort sheet** is intentionally a stub (IT-086). The pill reveals the
  legacy chip rows; the designed bottom sheet is IT-089.
- **Realtime may not push updates** without the publication fix noted in
  `workspace/phase-2-4-test-results.md` — reload to verify persistence.

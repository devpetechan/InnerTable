**Scope:** friendships + invite flow (IT-038–041), circle-scoped notes/comments (IT-042), notes hint (IT-043), feed lenses (IT-044), place tags, profile settings (IT-106).
**Design record:** ratings are network signal, notes are circle candor — see `workspace/v0.4.0-implementation-plan.md`.

1. Merge `feat/v0.4.0-friend-circles` → `main` and apply migrations **together** — the Phase 5+ frontend requires 0018/0019; the pre-0.4.0 frontend breaks against a migrated DB (entries.notes is dropped).
2. Apply `0017_friendships.sql`, `0018_entry_notes_and_comment_privacy.sql`, `0019_place_tags.sql` in order (0017 may already be applied).
3. Hard-refresh the deployed app (Pages cache).
